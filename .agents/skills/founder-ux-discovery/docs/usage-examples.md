# Usage Examples

## New discovery

“I have a product idea. Help me discover the UX.” Read relevant project memory, briefly identify what is already known, then ask the single highest-value question.

## Explore alternatives

“We understand the problem. Show several fundamentally different UX approaches.” Generate 3–5 concepts with different interaction models.

## Compare and decide

“Compare the concepts and recommend one.” Apply the decision framework and make an opinionated recommendation.

## Specify

“We chose the Personal Navigator model. Create the UX specification.” Produce the structured specification without premature visual styling.

## Continue later

“Continue the UX discovery.” Retrieve relevant memory, summarize known facts, decisions, and unresolved questions, and continue rather than restart.

## Record a decision

“Record the decision to choose Timeline over Chatbot.” Create a structured decision record and save it through the project’s configured memory workflow.
