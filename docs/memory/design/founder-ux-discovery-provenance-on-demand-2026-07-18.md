# Founder UX Discovery — provenance on demand

## Confirmed founder preference

Official source evidence does not need to be visible on every route action at first glance, as long as it is **clearly and immediately available** when a person wants to inspect it.

## UX implication (hypothesis)

Keep the default route focused on action and timing; make provenance a consistent, recognizable inspection path on every action. Source inspection must preserve publisher/review context and external-link clarity, rather than becoming a generic citation dump.
